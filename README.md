# Amazon-Clothing-Sales-Dashboard
Built an interactive Amazon Clothing Sales Dashboard in Microsoft Excel to analyze revenue, orders, AOV, and category performance. Cleaned and modeled data, created KPIs, charts, and slicers. Generated insights on trends and top products while fixing filter, relationship, and calculation issues for accurate reporting.

## Objectives
- Monitor overall sales performance  
- Analyze product category trends  
- Track key business KPIs  
- Improve decision-making through dashboards  

## Tools & Techniques
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- Data Cleaning  
- KPI Reporting  

## Key Metrics
- Total Sales  
- Total Orders  
- Average Order Value (AOV)  
- Top Selling Categories  
- Monthly Revenue Trends  

## Dashboard Features
- Interactive filters using slicers  
- Dynamic KPI summary cards  
- Monthly and category-wise trend analysis  
- Product performance comparison  
- User-friendly dashboard layout  

## Key Insights
- Identified highest-performing product categories  
- Observed monthly sales fluctuations  
- Highlighted top revenue-generating products  
- Analyzed customer buying patterns  

## Challenges Addressed
- Resolved slicer/filter issues  
- Corrected KPI calculation errors  
- Improved data accuracy and dashboard performance
